Detailed Description

Provides utility functions for tokenizing input lines and validating command structure.

The io class is responsible for parsing user input strings and validating the format of various elements such as potion names, ingredient lists, and complete commands.

Validates a token range to represent a correct ingredient list.

The expected form is: quantity name , quantity name , ... , quantity name (no comma after last one). Quantities must be positive numbers, and names must be alphabetic.

Validates a sequence of tokens as a properly formatted potion name.

Validates if a series of tokens represent a correctly formatted potion name.

Rules:

  • All words in the potion name must be alphabetic.
  • In the raw input line, there should be exactly one space between words.
  • Tokens must be contiguous.

startThe index of the first token of the potion name.
