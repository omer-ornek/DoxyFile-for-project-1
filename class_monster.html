Detailed Description

Represents a monster in the Witcher world.

Each monster has a name and a set of effective potions and signs that can be used to defeat it. The class provides functionality to learn about monsters, register their weaknesses, and handle encounters.

bool Monster::encounter_monster ( const string & name)

Attempts to encounter the specified monster using known effective potions or signs.

Handles the encounter with a monster and attempts to defeat it using known effective items.

Potions are consumed if available and known to be effective. A monster can be defeated either by an effective potion or by knowing effective signs.

Parameters
nameName of the monster to encounter.
Returns
true if the encounter is successful (monster can be defeated), false otherwise.

This function checks whether Geralt has an effective potion or sign for the monster. If an effective potion is used or there are known effective signs, Geralt wins.
