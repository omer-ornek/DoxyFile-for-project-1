void brew ( int index,
int end,
char ** tokens,
HashMap * potions,
HashMap * ingredients )

Attempts to brew a potion based on a known formula.

This function takes the potion name from the given tokens (spanning from index to end), looks up the potion in the potions HashMap, checks if the required ingredients are available in the inventory (from the potion's recipe), and if so, consumes the ingredients and increments the potion count in the inventory.

If the potion formula is unknown or ingredients are insufficient, it prints the appropriate message.

Parameters
indexThe starting index of the potion name tokens.
endThe ending index of the potion name tokens.
tokensTokenized input command.
potionsHashMap containing all known potions and their recipes.
ingredientsHashMap representing Geralt's current ingredient inventory.
