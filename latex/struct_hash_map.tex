\doxysubsection{Detailed Description}
Main structure representing the hash map. 

Represents the overall hash map data structure.

The map contains a table of buckets, each holding a pointer to a linked list of {\ttfamily \doxylink{struct_hash_node}{Hash\+Node}}s. Capacity defines the number of buckets. Size tracks the total number of key-\/value pairs stored in the map.
