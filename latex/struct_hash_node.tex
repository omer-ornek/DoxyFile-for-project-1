\doxysubsection{Detailed Description}
A single node in a linked list for separate chaining. 

Represents a single node (key-\/value pair) in the hash map\textquotesingle{}s linked list.

Each node contains a dynamically allocated key (string), a value pointer (can point to any data), and a pointer to the next node.
