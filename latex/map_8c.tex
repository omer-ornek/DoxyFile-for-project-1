\doxysubsection{Detailed Description}
A simple hash map implementation using separate chaining. 

This file implements a dynamically resizing hash map that stores (key, value) pairs. Keys are strings, and any pointer type can be used as the value. Collisions are resolved with linked lists (separate chaining). The table automatically resizes when the load factor exceeds a threshold (LOAD\+\_\+\+FACTOR).
