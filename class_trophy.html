Detailed Description

Manages trophy items obtained from defeating monsters.

Each trophy has a name and an amount. The class provides static methods to manage and query the trophy inventory globally.

Adds or removes a trophy from the inventory.

Updates the number of a specific trophy by a given amount.

If the trophy already exists, its count is increased by amount. If the new amount becomes 0, the trophy is removed from the map.

Parameters
nameThe name of the trophy.
amountThe amount to add (or subtract if negative). Default is 1.

If the trophy doesn't exist, a new one is added. If the new amount becomes 0, the trophy is removed.
